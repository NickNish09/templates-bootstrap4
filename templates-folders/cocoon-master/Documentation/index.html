Files Structure

  • Documentation – contains documentation on template editing and installation.
  • html (theme package).
    • assets (contains css , Scss , javascript, images and fonts)
      • css (contains theme custom css + css lib)
      • Scss (contains theme Scss)
      • fonts (contains icon fonts)
      • img (contains all theme images)
      • js (contains theme custom javascript + javascript lib )
    • index.html
    • aboutus.html
    • Blog.html
    • Contact.html
    • portfolio.html
    • sevices.html
    • elements.html




HTML Structure

This theme is a responsive. All of the information within the main content area is nested within a body element 
< body >

Page Content some code here 
< /body >
